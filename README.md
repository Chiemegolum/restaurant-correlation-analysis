# 🏘Restaurant Exploration and Correlation Analysis

## 🔍Project Overview
This is an exploratory analysis into 8,368 restaurants to discover what drives their revenue, as well as the top performing restaurants. The project depicts the cleaning workflow and well-detailed insights markdowns in Python.

## ⏳Project Goal
To clean and prepare the dataset for analysis. To identify the top 5 earners, the most rated restaurants, and as well as the correlation of the features to the revenue.

## 🛠Tools Used
- Python (Pandas, Matplotlib, Seaborn), Jupyter Notebook
